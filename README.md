# Learn Nextjs

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Create a migration script

```sh
npx drizzle-kit migrate
# khi có cập nhập cột mới hoặc type thì chạy 2 dòng dưới.
npx drizzle-kit generate
npx drizzle-kit push
```
